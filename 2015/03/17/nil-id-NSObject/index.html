nil&id&NSObject

发表于 2015-03-17 | 分类于 iOS

  • nil：空对象,发送消息安全

  • NSNull：值为 空 的对象

  • NULL：C的的空指针

  • id 可指向任何OC类型的指针,而 void * 表示C语言中可指向任何类型的指针

  • NSObject是OC的基类,但并不是所有OC对象都继承NSObject,例如NSProxy

  • NSOjbect还可以是协议,只不过同名罢了
